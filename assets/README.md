# Assets Directory

## Objetivo
Este diretório será a base futura dos recursos visuais e auxiliares do RestPK.

## Função deste diretório
Este diretório deverá concentrar, futuramente:
- recursos visuais do projeto
- arquivos de apoio ligados à identidade visual
- materiais auxiliares usados por cliente e mapa
- organização técnica dos assets do projeto

## Estado atual
Nesta etapa, este diretório ainda está em preparação.
O objetivo atual é apenas registrar com clareza sua função dentro do projeto.

## O que ainda não entra
- biblioteca visual completa
- sprites finais do projeto
- interface final
- grande volume de recursos gráficos
- organização final dos assets

## Observação
Este arquivo existe para evitar ambiguidade sobre a finalidade do diretório `assets/` antes do bootstrap técnico real.
