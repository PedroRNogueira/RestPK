# Server Directory

## Objetivo
Este diretório será a base futura do servidor do RestPK.

## Componente planejado
A base planejada para este diretório é o **Canary**.

## Função deste diretório
Este diretório deverá concentrar, futuramente:
- base do servidor
- arquivos relacionados à lógica principal do jogo
- estrutura ligada a scripts e comportamento do mundo
- organização técnica da parte servidor do projeto

## Estado atual
Nesta etapa, este diretório ainda está em preparação.
O objetivo atual é apenas registrar com clareza sua função dentro do projeto.

## O que ainda não entra
- importação completa da base do servidor
- build do servidor
- configuração final
- gameplay
- customizações profundas

## Observação
Este arquivo existe para evitar ambiguidade sobre a finalidade do diretório `server/` antes do bootstrap técnico real.
