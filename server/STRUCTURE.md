# Server Structure

## Objetivo
Este documento registra a organização inicial esperada para a área do servidor do RestPK.

## Estrutura inicial esperada
Nesta fase, a área do servidor deve ser entendida como o espaço futuro para:
- base principal do servidor
- scripts de jogo
- arquivos de configuração
- organização técnica da lógica do mundo

## Papel dessa estrutura
Esta organização servirá, futuramente, para:
- separar claramente a parte servidor do restante do projeto
- permitir evolução incremental da lógica do jogo
- facilitar a localização de arquivos importantes
- reduzir ambiguidade durante o bootstrap técnico do servidor

## O que ainda não entra
- estrutura final importada do Canary
- refatoração de arquivos reais
- gameplay completo
- configuração final de execução
- customizações profundas

## Observação
Este documento ainda não representa a estrutura definitiva do servidor.
Ele apenas registra a organização inicial esperada antes da integração técnica real.
