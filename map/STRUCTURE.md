# Map Structure

## Objetivo
Este documento registra a organização inicial esperada para a área de mapas do RestPK.

## Estrutura inicial esperada
Nesta fase, a área de mapas deve ser entendida como o espaço futuro para:
- arquivos de mapa
- estrutura espacial inicial do projeto
- organização técnica da base de mapas
- recursos ligados à construção das áreas jogáveis

## Papel dessa estrutura
Esta organização servirá, futuramente, para:
- separar claramente a parte de mapas do restante do projeto
- permitir evolução incremental dos mapas do jogo
- facilitar a localização de arquivos importantes
- reduzir ambiguidade durante o bootstrap técnico dos mapas

## O que ainda não entra
- mapa completo do jogo
- cidade inicial final
- rotas completas
- distribuição extensa de conteúdo
- expansão completa do mundo

## Observação
Este documento ainda não representa a estrutura definitiva dos mapas.
Ele apenas registra a organização inicial esperada antes da integração técnica real.
