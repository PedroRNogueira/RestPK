# Map Directory

## Objetivo
Este diretório será a base futura dos arquivos de mapa do RestPK.

## Componente planejado
A base planejada para este diretório é o uso do **Remere's Map Editor**.

## Função deste diretório
Este diretório deverá concentrar, futuramente:
- arquivos de mapa
- estrutura espacial inicial do projeto
- organização técnica da base de mapas
- recursos ligados à construção das áreas jogáveis

## Estado atual
Nesta etapa, este diretório ainda está em preparação.
O objetivo atual é apenas registrar com clareza sua função dentro do projeto.

## O que ainda não entra
- mapa completo do jogo
- cidade inicial final
- rotas completas
- distribuição extensa de conteúdo
- expansão completa do mundo

## Observação
Este arquivo existe para evitar ambiguidade sobre a finalidade do diretório `map/` antes do bootstrap técnico real.
