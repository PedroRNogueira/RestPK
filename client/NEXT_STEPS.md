# Client Next Steps

## Objetivo
Este documento registra os próximos passos esperados para a área do cliente do RestPK.

## Próximos passos imediatos
Os próximos passos da área do cliente devem ser:

1. manter a documentação do cliente coerente com o projeto
2. preparar a futura entrada da base do OTClient
3. registrar claramente o papel da interface e apresentação do jogo
4. evitar qualquer expansão desorganizada da parte cliente

## O que deve orientar essa fase
- foco em organização técnica
- clareza sobre o papel do cliente
- separação entre documentação e implementação real
- preparação gradual para o bootstrap técnico do OTClient

## O que ainda não entra
- importação completa da base do cliente
- alteração profunda em código-fonte
- implementação de gameplay completo
- interface final
- sistemas avançados do jogo

## Observação
Este documento ainda não inicia a integração técnica real do cliente.
Ele apenas registra o próximo recorte lógico de avanço da área cliente dentro do projeto.
