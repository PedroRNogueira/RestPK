# Setup

## Objetivo
Este documento registra o checklist inicial de ambiente do projeto RestPK e organiza a preparação técnica antes da integração real da stack.

## Status atual
- Repositório estruturado
- Documentação base em andamento
- Stack planejada definida
- Instalação completa da stack ainda não iniciada

## Ferramentas base
- Git
- GitHub
- Editor de código recomendado
- Terminal para operações de Git e setup
- Acesso ao repositório remoto

## Stack planejada
- **Servidor**: Canary
- **Cliente**: OTClient mehah / OpenTibiaBR
- **Editor de mapa**: Remere's Map Editor
- **Linguagens principais**: C++ e Lua

## Checklist inicial do ambiente
- Git instalado e funcional
- Acesso ao GitHub funcionando corretamente
- Repositório local sincronizado com o remoto
- Editor preparado para trabalhar com arquivos Markdown, Lua e C++
- Estrutura inicial do projeto criada no repositório
- Documento de regras do agente disponível no projeto

## Dependências futuras
- Ambiente necessário para compilar ou executar o servidor
- Ambiente necessário para trabalhar com o cliente
- Ferramentas para edição e testes de mapa
- Dependências específicas da stack OT que serão documentadas na próxima etapa

## O que ainda não entra nesta etapa
- Instalação completa do Canary
- Instalação completa do OTClient
- Instalação completa do RME
- Integração entre servidor, cliente e mapa
- Implementação de gameplay

## Próxima etapa esperada
A próxima etapa deve detalhar a preparação técnica real da stack OT, incluindo o plano de bootstrap de servidor, cliente e mapa.
