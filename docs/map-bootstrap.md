# Map Bootstrap

## Objetivo
Este documento registra a preparação mínima esperada para a base de mapas do RestPK.

## Editor planejado
O editor de mapa planejado do projeto é o **Remere's Map Editor**.

## Função desta etapa
Nesta etapa, o foco ainda não é construir o mundo completo do jogo.
O foco é apenas preparar o entendimento técnico inicial da base de mapas.

## O que deve ser definido nesta fase
- papel do RME dentro do projeto
- função dos mapas na estrutura do jogo
- relação entre mapa, servidor e cliente
- estrutura inicial onde os arquivos de mapa ficarão no repositório

## O que ainda não entra
- mapa completo do jogo
- grande quantidade de áreas
- polimento visual avançado
- distribuição extensa de conteúdo
- expansão completa do mundo

## Resultado esperado
Ao final desta etapa, o projeto deve ter uma referência documental clara sobre o papel dos mapas e sobre o que será necessário para preparar sua base futuramente.

## Observação
Este documento ainda não é um guia de edição de mapas.
Ele serve apenas para alinhar o bootstrap inicial da base de mapas dentro do projeto.
